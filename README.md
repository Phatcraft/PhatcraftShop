# Phatcraft Shop

Phatcraft Shop là 1 project về 1 trang web shop bán hàng của tôi.\
Được viết bằng PHP

# Setup cơ bản
  + ### Cài đặt PHP, MySQL
> [!CAUTION]
> Yêu cầu tối thiểu: `php >= 8.2', 'mysql >= 15.1'

  + ### Export file `phatcraft_shop.sql` vào `mysql`
  ````bash
  $ mysql -u <username> -p < phatcraft_shop.sql
  ````

  + ### Chạy server
  ````bash
  $ php -S <host>:<port>
  ````
  > Các `<host` và `<port>` bạn có thể tự lựa chọn
