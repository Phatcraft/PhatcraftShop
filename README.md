# Phatcraft Shop

Phatcraft Shop là 1 project về trang web shop bán hàng của tôi. <br>
Tại đây, người dùng có thể mua hàng với giá cả phải chăng nhất. <br>
Trang web được viết chủ yếu bằng PHP.
> Đây chỉ là 1 project, không phải là sản phẩm chính thức

# Setup cơ bản
  + #### *Cài đặt PHP, MySQL*
> [!CAUTION]
> Yêu cầu tối thiểu: `php >= 8.2', 'mysql >= 15.1'

  + #### *Export file `phatcraft_shop.sql` vào `mysql`*
  ````bash
  $ mysql -u <username> -p < phatcraft_shop.sql
  ````

  + #### *Chạy server*
  ````bash
  $ php -S <host>:<port>
  ````
  > Các `<host` và `<port>` bạn có thể tự lựa chọn
